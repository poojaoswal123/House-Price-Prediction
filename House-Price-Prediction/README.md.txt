# 🏠 House Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts house prices using Machine Learning based on various property features such as area, bedrooms, bathrooms, parking, air conditioning, furnishing status, and more.

The project demonstrates the complete Machine Learning workflow including data preprocessing, exploratory data analysis (EDA), model building, evaluation, and saving the trained model.

---

## 🎯 Problem Statement

The objective of this project is to build a machine learning model that can accurately predict the selling price of a house based on its features.

---

## 📂 Dataset

* Dataset Source: Kaggle
* Number of Records: 545
* Number of Features: 13

Target Variable:

* **price**

Input Features:

* area
* bedrooms
* bathrooms
* stories
* mainroad
* guestroom
* basement
* hotwaterheating
* airconditioning
* parking
* prefarea
* furnishingstatus
